# EMFRONTIER LAB - 클라이언트 사이트

정책자금 신청을 위한 클라이언트 전용 웹사이트입니다.

## 🚀 실행 방법

```bash
npm install
npm run dev
```

서버가 http://localhost:3000 에서 실행됩니다.

## 📱 주요 기능

- 회원가입 및 로그인
- AI 기반 SOHO 등급 진단
- 정책자금 자동 추천
- 진행상황 실시간 확인
- QR 코드 생성

## 🔑 테스트 계정

회원가입 후 사용하세요.
